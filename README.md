📌 Project Overview 

This project focuses on analyzing customer behavior to identify churn risk and credit default risk in a credit card portfolio. 

The objective is to build an end-to-end analytics pipeline to support portfolio monitoring and business decision-making. 

🎯 Business Problem 

Why are customers: 

Reducing spend? 

Becoming inactive? 

Defaulting on payments? 

The project answers these questions using behavioral and financial data.

## 📊 Data Sources 

- Bank churn dataset (churn behavior) 

- Uci credit dataset (repayment & delinquency)
  
Key data includes:

- Credit limit 
- Transaction activity 
- Utilization ratio
- Payment history 
- Delinquency (DPD)

🧠 Key Insights 

Customers inactive for 4+ months show highest churn risk  

Declining transaction activity is an early churn signal  

High credit utilization (>70%) strongly correlates with default  

Delinquency (DPD) is the strongest predictor of default risk  

Payment gaps indicate financial stress 

🚀 Business Recommendations 

Launch targeted retention campaigns for inactive customers  

Offer rewards & incentives to increase engagement  

Monitor high utilization customers proactively  

Strengthen collections for high delinquency accounts
